Objective
Desenvolver a funcionalidade de reconhecimento inteligente de equipamentos via imagem com IA (Google Gemini Vision), retornando exercícios sugeridos com instruções detalhadas, visualização de músculos no corpo humano (frente e costas), e integração completa no fluxo do app "Eu Me Treino". Toda a UI em português brasileiro. Usar a chave de API do Gemini que o usuário já forneceu.

Tasks

Blocked By: []
Details:
Solicitar ao usuário a chave de API do Gemini (não encontrada nos secrets atuais) e configurar como secret GEMINI_API_KEY
Instalar pacote @google/generative-ai no backend
Criar arquivo server/gemini.ts com client Gemini configurado usando a chave de API
Exportar instância do modelo gemini-2.5-flash (suporta visão/imagem)
Files: server/gemini.ts
Acceptance: Client Gemini configurado e exportando modelo pronto para uso com imagens
T002: Criar endpoint de reconhecimento de equipamento via imagem
Blocked By: [T001]
Details:
Criar endpoint POST /api/vision/identify em server/routes.ts
Aceitar imagem em base64 no body da requisição
Enviar para Gemini Vision com prompt otimizado que:
Identifica equipamento(s) de academia/exercício na imagem (incluindo objetos adaptáveis como garrafas, cadeiras, escadas, mochilas)
Retorna JSON estruturado com: equipment_name, confidence (0-100), suggested_exercises[]
Cada exercício contém: name, description (instrução detalhada em pt-BR), muscles_primary[], muscles_secondary[], sets, reps, rest_seconds
Fallback: se confiança < 70%, retornar flag low_confidence: true para o frontend tratar
Tratamento de erros robusto
Files: server/routes.ts
Acceptance: Endpoint retorna JSON com equipamento identificado e exercícios sugeridos
